# 🤖 Real-Time Object Detection with YOLO11

A computer vision project built during the **VisionX AI Robotics Bootcamp**

## 🎯 Features
- Real-time webcam detection using YOLOv11
- Detects cell phone position: LEFT / CENTER / RIGHT
- Frame divided into 3 zones with bounding boxes

## 🛠️ Tech Stack
- Python
- OpenCV
- Ultralytics YOLO11

## ⚙️ Installation
pip install -r requirements.txt

## ▶️ Run
python workshop.py
